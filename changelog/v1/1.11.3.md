# NaturalModels 1.11.3

## Notice
- It will be helpful to check 'template texture' docs to ignore resource pack generation.
- There are some deprecated API in ModelManager and PlayerManager.

## Add
- allow textures in NaturalModels/players with [template texture](https://github.com/toxicity188/NaturalModels/wiki/Template-texture)
- add player/monster nametag (ptag_, mtag_)
- ModelRenderer API
- more NaturalModels API
- improved reload speed
```
NaturalModels.modelOrNull("name"); //Gets nullable model renderer
NaturalModels.model("name"); //Gets optional model renderer
```

- avoid classloader conflict of Adventure in Paper.
- tracker remove refreshment
- minor optimized memory cost
- animation script
